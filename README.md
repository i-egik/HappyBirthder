Программа переносится на Spring-boot.
Пока что реализовано консольное управление.
При первом запуске нужно будет добавить людей, в дальнейшем программа будет автоматически подгружать их из файла
При каждом последующем запуске, программа загружает информацию из текстового файла и автоматически отправляет поздравление людям, у которых ДР "сегодня"
Для управления, воспользуйтесь подсказкой-инструкцией.
